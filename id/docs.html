Referensi Lengkap v2.0

Selamat datang di panduan utama untuk Audio XML (AXML). Spesifikasi ini mendefinisikan format berbasis teks yang kuat untuk sintesis dan komposisi musik di web.

Apakah Anda seorang desainer suara yang memahat synth melalui kode atau seorang komposer yang membangun aransemen kompleks, AXML memberi Anda kendali presisi atas setiap frekuensi dan ketukan.

account_tree Hierarki Root

File AXML yang valid harus mengikuti urutan elemen yang tepat ini. Engine memprosesnya secara berurutan untuk membangun graf audio.

<?xml version="1.0" encoding="UTF-8"?>
<axml version="1.0">
  <metadata> ... </metadata>
  <instruments> ... </instruments>
  <patterns> ... </patterns>
  <tracks> ... </tracks>
</axml>

info Metadata Global

Metadata mendefinisikan konteks proyek. tempo sangat penting karena mendefinisikan bagaimana beat diterjemahkan ke detik.

Elemen Deskripsi
<title> Nama tampilan proyek.
<artist> Pembuat proyek atau desainer suara.
<tempo> Beats Per Minute (60.0 - 250.0). Rata-rata: 120.
<key> Nada dasar musik (misalnya, C, Am, F#m).
<time-signature> Struktur ritme (misalnya, 4/4, 3/4). 
<metadata>
  <title>Cyberpunk City</title>
  <artist>AXML Master</artist>
  <tempo>128</tempo>
  <key>Am</key>
</metadata>

waves Tipe Osilator

Atribut type pada <instrument> menentukan bentuk gelombang dasar. AXML saat ini mendukung 11 tipe berbeda:

Nama Tipe Gelombang Matematis / Model Fisik
sine Gelombang sinus murni (Hanya frekuensi fundamental).
square Gelombang kotak (Kaya akan harmonik ganjil).
sawtooth Gelombang gigi gergaji (Semua harmonik, sangat terang).
triangle Gelombang segitiga (Harmonik lembut).
kick Model fisik: Sapuan sinus + Peluruhan eksponensial cepat.
snare Model fisik: White noise + Lowpass filter pada 1.2kHz.
hihat Model fisik: White noise + Highpass filter pada 8kHz.

gesture Envelope ADSR

Envelope volume sangat penting untuk mendefinisikan "bentuk" suara. Semua nilai dalam detik, kecuali sustain (0.0 hingga 1.0).

Atribut Nama Lengkap Perilaku
attack Attack Time Detik untuk mencapai volume puncak.
decay Decay Time Detik untuk turun ke level sustain.
sustain Sustain Level Faktor volume (0-1) yang ditahan selama nada aktif.
release Release Time Detik untuk memudar ke hening setelah nada berakhir.

auto_fix_high Rak Efek (FX)

AXML Studio v2.0 memiliki rak efek yang dapat dirangkai untuk setiap instrumen.

Efek Atribut Rentang
Pan pan -1.0 (Kiri) hingga 1.0 (Kanan)
Distortion distortion 0.0 hingga 1.0
Chorus chorus 0.0 hingga 1.0
Bitcrush bitcrush 0.0 hingga 1.0
Reverb reverb 0.0 hingga 1.0

keyboard Shortcut Keyboard

Percepat workflow desain suara Anda di Studio:

  • Space : Play / Stop Proyek Saat Ini
  • Ctrl + S : Simpan ke Penyimpanan Lokal
  • Ctrl + I : Format Otomatis Kode XML
  • Tab : Identasi Kode (2 spasi)
